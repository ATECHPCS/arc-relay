Connection Examples

Connect AI tools to this MCP server through MCP Wrangler.

mcp-sync CLI (Recommended)

The easiest way to connect. Handles config files, auth headers, and per-project management automatically. 

mcp-sync add {{.Server.Name}}
First-time setup & install 
# Install (Linux)
sudo curl -fsSL https://github.com/JeremiahChurch/mcp-sync/releases/latest/download/mcp-sync-linux-amd64 \
  -o /usr/local/bin/mcp-sync && sudo chmod +x /usr/local/bin/mcp-sync

# Install (macOS Apple Silicon)
sudo curl -fsSL https://github.com/JeremiahChurch/mcp-sync/releases/latest/download/mcp-sync-darwin-arm64 \
  -o /usr/local/bin/mcp-sync && sudo chmod +x /usr/local/bin/mcp-sync

# First-time setup — enter this wrangler's URL and your API key
mcp-sync init
Claude Code (.mcp.json) — manual setup

Add to .mcp.json in your project root:

{
  "mcpServers": {
    "{{.Server.Name}}": {
      "type": "http",
      "url": "{{.BaseURL}}/mcp/{{.Server.Name}}",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
