Connect Claude Desktop

Add your MCP servers to Claude Desktop as custom connectors. Authentication is handled automatically via OAuth - no API keys needed.

How it works
  1. Copy a connector URL below
  2. In Claude Desktop: profile icon > Settings > Connectors > Add custom connector
  3. Paste the URL and click Add
  4. A browser window opens - log in and click Approve
  5. Done - the server's tools are now available in Claude Desktop
